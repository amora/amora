
The following mouse commands are supported:
  1) Mouse moving and click: use cellphone joystick
  2) Mouse drag: press key '1' and use joystick
  3) Right mouse button: keys '3'
  4) Mouse wheel: keys '7' (scroll up) and '*' (scroll down)

And also some keyboard shortcuts:
  ENTER: key '5'
  Arrow key left: key '4'
  Arrow key right: key '6'
  F5: key '8'
  SPACE: key '0'
  ESC: key '9'
  F (mplayer fullscreen): key '#'
  DEL: clear key
  SCREENSHOT: use key '2'
